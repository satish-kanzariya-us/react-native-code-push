# @logicwind/react-native-code-push

code push library for react natiev

## Installation


```sh
npm install @logicwind/react-native-code-push
```


## Usage


```js
import { multiply } from '@logicwind/react-native-code-push';

// ...

const result = multiply(3, 7);
```
